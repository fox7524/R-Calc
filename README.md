# English R-Calc:
## How to use via terminal?

### For Mac:

#### 1
Put this file in document path that you are not going to change documents for example.
#### 2
Open terminal
#### 3
Insert this command in the terminal: nano ~/.zshrc
#### 4
With the arrow keys add this command at that file(recommended place is the end of the file):
alias direnc(<— you can change the alias)= ‘python3 (paste your file path here)direnc.py'
#### 5
Press ctrl + O and press enter
#### 6
Press ctrl + X and quit from file
#### 7
Activate the alias with this command: source ~/.zshrc

# Turkish R-Calc:

#### 1
